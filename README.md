normalize.Scss
=============
An up-to-date SCSS version of normalize.css

This way you can include it in your COMPASS projects as a submodule, getting updates to normalize.css with a simple: 
$ git submodule foreach git pull
and minify it for your production environment.

How to submodule:
http://git-scm.com/book/en/Git-Tools-Submodules
How to render on your development environments:
compass -e development
How to render for your production environment:
compass -e production
